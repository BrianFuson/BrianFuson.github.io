Cat Zoetrope

This is a project I created in my GRDS 471 class. Looking into personal branding, the class had to come up with 3 things we are and 3 things we are not. I landed on I am Imaginative, Disciplined, and Experimental as well as I am not Steady, Modern, and Formal. The next step was a set of personal icons (of my cat). The icon set started as fun pictures of my cat, Frank, and I then mixed them with a font I created, Knowlton Values, to have vector ASCII icons. The next step I took to push this project further was experimenting with motion. I selected a Zoetrope record. This mixes the rpm of the record player with the frame rate of a camera to result in an animation. I figured this out through a lot of trial and error and the end result is exactly what I was wanting.

This portfolio website is coded in HTML, CSS, and JavaScript by Brian Fuson.
